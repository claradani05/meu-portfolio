Meus Projetos

WOD SYSTEM

Este projeto é um sistema de cadastramento de WOD (Workout of the Day) desenvolvido para um box de CrossFit. A ideia principal é facilitar a organização e o acompanhamento dos treinos para alunos e instrutores.

Plataforma Ecommerce

Este projeto é uma plataforma de e-commerce focada na venda de ingressos online. Ainda em seus estágios iniciais de desenvolvimento, a aplicação visa criar um sistema robusto e intuitivo para a compra e venda de entradas para eventos, shows e outras atividades.

Jogo da Cobrinha

Uma recriação do clássico jogo Snake, utilizando Python e a biblioteca Pygame. Este projeto demonstra habilidades em programação orientada a objetos e lógica de jogo.
